# jobrad-chat-coding-task

## architecture
ChatApp_Architecture.drawio.pdf - system design of my chat app
ChatApp_Classes.pdf - the objects needed for the chat app
ChatApp_Descriptions.pdf - detailed descriptions for the questions of the coding task

## backend
The backend is written in C# using .NET9 Framework.

## frontend
The frontend is built using React vite as a single page application.
It will be running on localhot in port 5174

http://localhost:5174 for the customer view
http://localhost:5174/agent for the agent view

## Makefile
Run `make dev` to start backend and frontend
